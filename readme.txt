Split order by Category for Woocommerce" allows splitting an order into separate orders based on category.

== Description ==

"Split order by Category for Woocommerce" allows splitting an order into separate orders based on category. This Plugin can be used to split an order automatically into multiple orders based on the category of the products in the cart. The customer will receive different order ids for their ordered cart. 

Note: This is a free variant of Split order by category for woocommerce. 

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the /Plugins/ screen in WordPress Split by Category
2.1. Select Default setting:
A) Default Condition
When the condition is Default then the order is split irrespective of any attribute. Example: If an order has 4 products then the order is split into 4 different orders no matter what attribute is chosen.

==Features of Plugin

Helps to manage the order as per the selected category
As this plugin splits the order as per the selected category it makes the management of the order for admin much quicker and easy.

The customer also receives different order Ids according to the selected category existing in the order
When buyers place an order with multiple items in the cart then, separate order ids will be generated.

Separate order ids will be generated at the back-end as per the category selected
Separate order ids will be generated in the back-end as well when a buyer places an order with having multiple products in the cart.

 == Frequently Asked Questions ==

= Will the customer pay multiple times?
Payment will be made only once for a complete order amount, later the order splits as per the items in cart.

=How is the payment handled when we connect with a Payment Service Provider.
The payment is being taken before the order is split so the complete order amount is taken from the pay.

=What is the difference between Paid plugin Split Order On Checkout Pro, Advanced Split Order On Checkout, and split order by category?
Split Order On Checkout Pro Plugin splits the order into different ID’s according to conditions set as per the assigned attribute at the backend.

Advanced Split Order On Checkout Plugin splits the order into different ID’s according to assigned attribute as per different conditions. This Plugin also enables to create various admin users with restricted roles.

Split Order Based on Category to split the order into different orders based on the categories of products in that particular order.

=Is it compatible with all payment gateways? Like offline and online payment methods
Yes, Our Plugin is compatible with offline and online payment gateway. But Currently, this is supported by only the PayPal payment gateway. 

=Does the user make a one-time payment or is it multiple payments for the different orders created?
There is only one payment taken for all the items in the cart, once the order is placed it split into different IDs.

=Does this Plugin support Wordpress latest version?
Ans. Yes, it supports WordPress version 5.6 (Latest version)

=Is this a free plugin for Split order by category for woocommerce ?
Yes, this product is a free variant of Split order by category. This plugin contains only default features. Default Features means When the condition is Default then the order is split irrespective of any attribute. Example- If a customer adds four products into the cart, then four orders will be created as per the default attribute.
